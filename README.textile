h2. About oUF_Quaiche

This is my own custom oUF layout. As with all oUF layouts, this addon requires oUF to function.

Provides the following Unit Frames
* Player
* Target
* Target of Target 
* Focus
* Focus Target
* Pet
* Party
* Party Pets
* Raid
* Raid Pets

h2. Required and Optional Dependencies

You must download and install oUF to use this layout:
"http://www.wowinterface.com/downloads/info9994-oUF.html":http://www.wowinterface.com/downloads/info9994-oUF.html
